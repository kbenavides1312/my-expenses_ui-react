# my-expenses.ui-react

## Development

1. Create react app

```
npx create-react-app my-expenses-ui
```

1. Add bootstrap style

```
npm install react-bootstrap bootstrap
```

1. Setup Routes

```
npm install react-router-dom
```

1. Setup Redux

```
npm install redux react-redux
```
